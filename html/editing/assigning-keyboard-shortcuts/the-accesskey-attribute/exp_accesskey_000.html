Set the accesskey value to:

  • This exploratory test allows you to quickly specify an accesskey value and test whether it works.
  • The large box has an accesskey attribute, which is set to the value you type into the input. You can then try to activate the box by pressing the same key with the modifier keys required to activate the shortcut. If you are successful, the large box will turn green.
  • Note that some keyboards interact with the modifier keys pressed, and prevent the use of a given key. It may help you to turn on an onscreen keyboard, so that you can see (to a degree) what is happening.
