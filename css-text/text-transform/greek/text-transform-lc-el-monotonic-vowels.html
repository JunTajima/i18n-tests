The need for contextual information to establish where accents belong when converting from all-caps uppercase to lowercase makes it unlikely that text-transform:lowercase is able to do that job correctly. Therefore we test here whether capital Greek letters that would appear in titlecased text convert to their lowercase correlates. This should be straightforward.

It is also assumed that iota and upsilon never appear in titlecase position with dialytika together, so there are no tests for that.

The test contains pairs of uppercase monotonic characters; text-transform:lowercase is applied to the right-hand character in each pair. The red test data is overlaid by black reference results.
