Change the font:

  • The proprietary -webkit-text-stroke is set to 4px pink. The standard text-stroke property is not supported by browsers.
  • The 'skip' directive checks that -webkit-text-stroke is supported. If not, the test is invalid.
  • The default test uses the Noto Naskh Arabic webfont, but the font change control allows you to test whether particular fonts make a difference to the outcome.
